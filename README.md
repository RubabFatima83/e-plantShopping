# Paradise Nursery - E-Plant Shopping

Welcome to **Paradise Nursery**, your one-stop online shop for a wide variety of plants, ranging from air-purifying and medicinal plants to aromatic and low-maintenance options. This project is a **React & Redux-based e-commerce application** for managing plant products and cart functionality.

---

## Project Overview
**Paradise Nursery** allows users to:
- Browse plants by category.
- View plant details including name, description, image, and price.
- Add plants to a cart and see the number of items in the cart dynamically.
- Update quantities or remove items from the cart.
- See the total price of items in the cart.
- Continue shopping or proceed to checkout (checkout functionality can be extended in future updates).

---

## Features
- Product listing with categories:
  - Air Purifying Plants
  - Aromatic Fragrant Plants
  - Insect Repellent Plants
  - Medicinal Plants
  - Low Maintenance Plants
- Dynamic cart with:
  - Item quantity control
  - Total cost calculation
  - Continue shopping button
- Responsive navigation bar with cart item count
- Easy-to-read product card layout with images and descriptions

---

## Technologies Used
- **Frontend:** React.js, CSS3
- **State Management:** Redux (Redux Toolkit)
- **Version Control:** Git & GitHub
- **Other Libraries:** React-Redux
